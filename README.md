# 👤 Desafio Perfil Social

## 📌 Sobre o projeto

Projeto desenvolvido em HTML com o objetivo de criar uma página de perfil social, apresentando informações pessoais e links para redes sociais.

## 🛠️ Tecnologias utilizadas

* HTML5

## 🎯 Funcionalidades

* Exibição de informações pessoais
* Inserção de foto de perfil
* Lista de links para redes sociais
* Estrutura organizada e intuitiva
